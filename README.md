# Employee-Attrition-Prediction-Retention-Insights
Built an end-to-end ML pipeline to predict employee attrition and  optimize decision thresholds using ROC analysis.
Machine learning tool to predict employee attrition and identify key drivers using Python and Streamlit.
## Features
- Built **ML pipeline** for **1,000+ employee records** (data cleaning, encoding, class imbalance handling).
- Trained **Random Forest model**, achieving **85% accuracy**.
- Identified top attrition drivers: **overtime, job role, tenure, and income**.
- Developed **interactive Streamlit app** to predict attrition risk for HR decision-making.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Streamlit, Matplotlib
